# 原创开题方案 · 备选两则

> 与《扑街文评论区求生》（方向一·元叙事数据流）配套，提供两个**完全不同方向**供对比挑选。
> 方向二＝轻松摆烂喜剧（i 人视角）；方向三＝高冷经济奇幻（商战悬疑底）。
> 三个方案的脑洞机制、调性、受众刻意岔开，避免同质。

# 方向二

## 《全员病娇追我，但我只想把他们静音》

> 备选书名：
> - 《社恐穿进病娇文，我只想躺平当个NPC》
> - 《我把所有人的好感度调成了静音》
> - 《e 人围猎，i 人摆烂》

**类型标签**：穿书 / 病娇群像 / 摆烂轻喜 / 反向恋爱脑 / 双向救赎
**受众**：吃病娇独宠、但腻了傻白甜恋爱脑女主的读者；社恐 / i 人强共鸣
**一句话卖点**：
> 她穿进一本"全员病娇为女主疯魔"的小说，成了那个被所有人偏执爱着的女主——可她偏偏是个**极致社恐 + 恋爱脑绝缘体**，只想当个透明 NPC 躺平保命。好在她头顶有个"**好感度调节台**"，能把追求者的好感一键**静音**。但她很快发现：好感不会消失，只会**积压反弹**——静音得越狠，黑化炸得越响。

### 核心脑洞（机制 + 代价）
- **好感度调节台**：女主能看见并调节每个人对她的好感数值，可"静音 / 调低 / 暂时拉黑"。
- **外挂有代价（守恒定律）**：好感是**守恒**的——被她调低/静音的好感不会蒸发，而是积压成"**淤积值**"。淤积值满 → 触发"病娇暴走"，对方黑化、且越压越猛。**她越想清净，越被逼到墙角。**
- **可验证**：所有数值明牌在面板上，读者和女主都看得见淤积进度条，知道"再静音第三次他就要疯"。
- **反向爽点**：别人是"努力攻略让人爱我"，她是"拼命降温让人放过我"——**目标反着来，喜剧张力全在这。**
- **弧光**：从"把所有人当麻烦、只想静音躺平"→ 发现"静音 = 慢性谋杀对方"→ 学会真正去**面对和回应**一段感情。i 人的成长是"从逃避到敢于被看见"。

### 人物
- **女主 · 温吞**：人如其名，钝感、社恐、班味十足的摆烂体质，遇事先想"能不能不参与"。毒舌内心戏 + 表面乖巧的反差是主要笑点来源。
- **男主 · 厉决**：原书头号病娇，占有欲天花板。被她静音第一次时一脸"我的手机是不是坏了"，发现是被人为调低好感后，**偏执地想搞明白"为什么她在躲我"**——越躲越上头。看点：他得学着"被讨厌也不发疯"，淤积值才不爆。
- **配角群像**：阴郁竹马、献祭型暗恋者、表面阳光实则掌控狂……每人一条淤积进度条，构成"病娇生态多线程管理"的爽点与笑点。
- **系统 · 静音管家**：一个只会播报"淤积值 +1，建议宿主跑路"的摆烂电子搭子。

### 热梗融入（载体＝女主毒舌内心 + e/i 人次元壁）
i 人 / e 人 / 班味 / 硬控 / 退退退 / 那咋了 / 已读乱回 / 搭子 / 公主请上车 / 显眼包 / 精神内耗 / 社交电量 / 那我走 / 尊嘟假嘟。
> 用法：女主是极品 i 人，原住民全是疯批 e 人，"社交电量""退退退"这类梗天然从她内心戏里长出来；梗服务"社恐 vs 围猎"的反差，不硬塞。

### 前三章钩子
1. **第1章**：睁眼即被全员宠成糖罐，温吞却只想找个角落充电。发现头顶调节台，一时手贱把男主好感**静音**——男主当场表情管理失效。章末：她瞥见男主头顶悄悄涨起一条红色【淤积值】。
2. **第2章**：她变本加厉群体静音，世界终于清净了……三天后，第一个被她静音的竹马**淤积爆表黑化**，把她堵在角落："你是不是……在躲我？"——规则砸脸：静音=养雷。
3. **第3章**：系统播报"全员淤积值集体逼近红线"，温吞被迫从"躺平静音"转向"亲手拆弹"。章末男主淡淡开口："别人可以静音，我不行。"——他主动把自己的好感**调回了最大声。**

### 文风样章（约 350 字）
> 温吞穿越后悟出的第一个道理是：这本书有毒。

她不过是想安安静静地坐在沙发角落，把今天的社交电量回一点血。

可是不行。

左边，俊美如雕塑的男人单膝跪在她脚边，正小心翼翼地替她换鞋，抬眼时眼神虔诚得像在供奉神明。右边，阴郁的少年抱着她掉的一根头发，宝贝得跟传家宝似的。门口还堵着三个，排队等着跟她"偶遇"。

温吞：……

【厉决 对你的好感度：998 ｜ 已突破系统量程】

她头皮一麻，下意识抬手，在那行飘着的数字上，按下了一个只有她能看见的按钮——

【静音】

世界，瞬间清净了。

跪在脚边的男人动作一顿，俊脸上第一次浮现出一丝……茫然。仿佛精密运转的仪器，突然丢了信号。

温吞松了口气，瘫回沙发，在心里疯狂续上一口电：**爽。早该这么干了。退退退，都给我退。**

她没看见的是，男人垂下的眼底，那行被她调成静音的数字底下，正悄悄浮起另一条，暗红色的——

【淤积值：1 ／ 100】

【温馨提示：好感守恒，压抑会反弹哦~ 本系统建议宿主……要不还是跑吧。】

# 方向三

## 《心动是这座城市的硬通货》

> 备选书名：
> - 《情绪炼金师》
> - 《我靠收割心动续命，唯独读不懂他》
> - 《这座城的眼泪明码标价》

**类型标签**：现言奇幻 / 情绪经济 / 冷感美人 / 商战博弈 / 暗黑唯美（非穿书·非系统·原生世界观）
**受众**：吃高冷设定、美学调性、双强博弈、冷 CP 破冰的读者；想看"有脑洞但不沙雕"的细糠党
**一句话卖点**：
> 在这座城市，**情绪可以被实体化提炼成"情绪石"**，是流通的货币与能源——心动最贵，悲伤次之，愤怒按吨甩卖。而她，是天生感受不到自己情绪的"**零感者**"，正因如此，她能冷静地收割、提纯别人的情绪，成了顶级的情绪操盘手。她用别人的心动续命，自己却从未心动过——直到遇见一个，她**读不出任何情绪**的男人。

### 核心脑洞（机制 + 代价）
- **情绪经济**：情绪能被提炼成"情绪石"，明码标价、可交易、可作能源。豪门财阀垄断"情绪矿脉"，底层人靠贩卖情绪糊口。**情绪价值是这个世界字面意义上的货币。**
- **女主体质·零感者**：感受不到自身情绪，所以不会被情绪反噬，能极限提纯他人情绪——顶级炼金天赋。
- **外挂有代价**：她**没有自己的情绪储备，必须持续吸取外界情绪才能活**（不吸会"情绪枯竭"而亡）。她在用别人的心动续命——**越强大，越空心。**
- **可验证**：每种情绪有公开汇率、纯度评级、储量表，博弈全程数据透明，像一套金融系统。
- **男主·反侦察体质**：她对全世界的情绪一目了然，**唯独读不出他**。这个"读不到"成了她的执念——她第一次，**想知道一个人对自己到底是什么感觉**，甚至第一次"想要心动"。
- **弧光**：从"把情绪当数字、把人当矿"的空心操盘手 → 在那个读不懂的人面前，**第一次长出属于自己的情绪**，从"收割爱"到"学会爱与被爱"。**（把'万人迷收爱意值'升级为可验证经济系统，并给主角'感受不到爱'的真实软肋。）**

### 人物
- **女主 · 沈漏**：冷感、精准、奢靡，行走的情绪交易所。表面无懈可击，内里是一片"感受不到"的荒原。气质参考"把睡袍穿成战袍"的神性疏离。
- **男主 · 裴砚之**：财阀继承人 / 或反垄断的"情绪游侠"，情绪反侦察体质，是她唯一的盲区与唯一的执念。双强，互为对手又互为解药。
- **反派 · 情绪矿阀**：垄断矿脉、把底层情绪当韭菜收割的资本怪物——天然带社会批判与爽点（打资本）。
- **关键配角**：一个把"快乐"卖光、只剩麻木的底层少年；一个囤积"悲伤"自我沉溺的旧爱——每个人都是一种"情绪病"的具象。

### 热梗融入（载体＝世界观词条 + 冷台词反差）
情绪价值 / 上头 / 精神内耗 / 班味 / 牛马 / 搞钱 / 电子榨菜 / 恋爱脑 / 课题分离 / EMO / 提供情绪价值 / 充电。
> 用法：这些词本就是当代情绪热词，直接当世界观术语用——"情绪价值"是字面汇率、"精神内耗"是一种可提炼的病、"充电"是续命。梗藏进设定里，不沙雕、反而高级。冷感女主偶尔蹦一句网感冷台词，制造反差。

### 前三章钩子
1. **第1章**：一场顶级情绪拍卖会，沈漏当众提纯一颗"绝版初恋心动"，技惊四座——立"情绪世界观 + 顶级操盘手"人设。章末她情绪储备告罄，第一次直面"再不吸取就会死"的代价。
2. **第2章**：她锁定下一个目标，却撞见裴砚之——全场唯一一个,她**读不出任何情绪数值**的人。盲区即危机，也是诱惑。
3. **第3章**：两人因一桩矿脉交易被迫合作/对立，她破天荒地**想从他身上"读出点什么"**而不得。章末她摸了摸自己的胸口，第一次困惑：那里好像，有什么东西，轻轻动了一下——【检测到未知情绪源：来源·宿主本人】。

### 文风样章（约 330 字）
> 这座城市最贵的东西，是心动。

沈漏站在拍卖厅最高处，指尖夹着一枚淡粉色的结晶，灯光穿过它，折射出细碎的、近乎甜蜜的光。

"绝版品。"她声音很淡，像在念一串与己无关的数字，"提纯纯度 99.7%，来源——一位老先生，对初恋长达六十年的心动。起拍价，三千万。"

台下倒吸冷气，举牌的手此起彼伏。在这座城，一颗如此纯度的"心动"，足够一个普通人续命十年。

而沈漏看着那枚为无数人疯狂的结晶，眼底没有一丝波澜。

她感受不到它的甜。

她是"零感者"——天生没有自己的情绪。正因如此，她能不被任何情绪干扰地收割、提纯它们，成了全城最顶尖的情绪炼金师。

别人靠心动悸动、靠悲喜活着。她靠把这些东西，**换成数字**。

落槌。三千八百万。

人群沸腾时，沈漏却悄无声息地按住了胸口——那里传来一阵熟悉的、冰冷的空。

【情绪储备：3% ｜ 警告：低于安全阈值，请及时吸取】

她唇角勾了一下，那弧度精准得没有温度。

又该"充电"了。

靠别人的心动活着的人，是体会不到心动的。

这一点，她早就习惯了。

——直到三天后，她遇见一个，她怎么也读不出情绪的男人。

# 三方向对比 · 怎么选

| 维度 | 方向一·弹幕求生 | 方向二·病娇静音 | 方向三·情绪经济 |
|---|---|---|---|
| 子类型 | 穿书·元叙事·数据流 | 穿书·病娇群像·摆烂喜剧 | 现言·原生世界·经济奇幻 |
| 调性 | 暗爽 + 网感轻喜 | 轻松搞笑为主 | 高冷 + 暗黑唯美 + 商战 |
| 文风 | 弹幕体、对镜头营业 | 毒舌摆烂、i 人内心戏 | 冷感克制、美学高级 |
| 热梗浓度 | ★★★（弹幕当规则，最浓） | ★★（i/e 人反差） | ★（情绪热词当术语，最克制） |
| 爽点 | 逆袭顶流 + 救书 | 病娇围猎 + 反差喜剧 | 双强博弈 + 冷 CP 破冰 |
| 上手难度 | 中（元叙事要控好） | 低（最好写、最稳） | 高（世界观要扎实） |
| 适合你若…… | 想要最强钩子、最网感、博爆款 | 想要轻松好写、稳定追读 | 想要高调性、长线深度、差异化 |

**我的建议**：
- 求**爆款相 + 平台脑洞偏好**：选**方向一（弹幕求生）**，钩子最硬、网感最强。
- 求**好写、稳、笑点密、新手友好**：选**方向二（病娇静音）**。
- 求**调性高、有记忆点、避开同质内卷**：选**方向三（情绪经济）**。

> 定下方向后，我可以立刻出该方向的【30 章分卷细纲】或【第一副本 4–10 章详纲】。
